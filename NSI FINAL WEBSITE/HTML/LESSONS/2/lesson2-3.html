Les variables

Les variables vont vous être ETRE très utiles,elles vont permettre de stocker des données et de les
utiliser plus tard.Il existe 2 types de variable:
👉Les chaines de caractères: str,on les repère dans les codes car elles sont entre des guillements.
👉Les nombres/chiffres: ce sont des nombres/chiffres entiers: Int (integer) ou des nombres à virgule: float,(flottant)

Comment déclarer les variables ?

Il y a quelques règles à respecter:

👉Nom de variable: Les noms de variables doivent commencer par une lettre (majuscule ou minuscule) ou un caractère de soulignement (_), suivi de lettres, chiffres ou caractères de soulignement.
✔️Valide : variable, _variable, variable1, Variable123
❌Non valide : 1variable, -variable, variable@abc

👉Sensibilité à la casse: Les noms de variables sont sensibles à la casse, ce qui signifie que "variable" et "Variable" sont considérés comme deux variables distinctes.

👉Mots-clés réservés: Les noms de variables ne doivent pas être des mots-clés réservés par Python. Par exemple, vous ne pouvez pas nommer une variable "if", "else", "while", etc. Voici la liste des mots-clés réservés en Python:
❌False, None, True, and, as, assert, async, await, break, class, continue, def, del, elif, else, except, finally, for, from, global, if, import, in, is, lambda, nonlocal, not, or, pass, raise, return, try, while, with, yield

👉Affectation de valeurs: Pour déclarer une variable, il suffit d'affecter une valeur à un nom de variable à l'aide de l'opérateur d'affectation (=). Python déterminera automatiquement le type de données de la variable en fonction de la valeur qui lui est attribuée.
✔️Exemple : variable = 42 (déclare une variable de type entier)
✔️texte = "Bonjour" (déclare une variable de type chaîne de caractères)


Si vous respectez toutes ces règles,ca devrait bien se passer !
